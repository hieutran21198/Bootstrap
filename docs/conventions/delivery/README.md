# Delivery conventions

Workspace-wide rules for how tracked work items are filed, progressed, and closed — the evidence-based delivery loop that ties Linear to the on-disk `docs/` record. Each rule below is a standalone file; this README is the index.

> **Scope**: every contributor and automated agent that files, updates, or closes work items in Linear for this workspace.
> **Status**: Active
> **Decided by**: [ADR-0017](../../adrs/0017-evidence-based-delivery.md)
> **Last reviewed**: 2026-07-04
