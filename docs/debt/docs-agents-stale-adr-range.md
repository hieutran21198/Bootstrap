# Generated root `AGENTS.md` states an outdated ADR range

> **Status**: Open

> **Last reviewed**: 2026-07-04

## What

The generated root `AGENTS.md` states an outdated ADR range ("ADRs currently run 0001–0012") while ADRs on disk run through 0018.

## Why it exists

`AGENTS.md` is generated by the init-deep skill and is not regenerated on every docs change, and no check flags the drift.

## Impact

- **Correctness**: misleads agents and contributors about the current ADR set.
- **Maintainability**: signals the generated knowledge base can silently go stale.

## Resolution

Regenerate `AGENTS.md` to reflect 0001–0018; consider a CI freshness check. `AGENTS.md` is generated and must not be hand-edited.
