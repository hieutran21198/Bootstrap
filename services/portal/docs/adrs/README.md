# Portal Architecture Decision Records

Portal-only decisions — choices that bind **this service** and do not constrain other services or shared packages. Workspace-wide decisions (Go SRP, code style, DDD/CQRS, UUIDv7, identity provider) live in [`docs/adrs/`](../../../../docs/adrs/); portal *implements* those, it does not re-decide them here.

> **Format, naming (`NNNN-kebab-case-title.md`), and lifecycle are authoritative in the workspace track** — see [`docs/adrs/README.md`](../../../../docs/adrs/README.md). Numbering is independent per service: portal ADRs start at `0001`.

Start a new ADR by copying [TEMPLATE.md](TEMPLATE.md).
