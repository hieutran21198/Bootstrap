# Portal debt

What portal still owes — architecturally-significant shortcomings in **this service** we have chosen to make visible. Routine `TODO` / `FIXME` nits stay inline in code; cross-cutting workspace or shared-package debt lives in [`docs/debt/`](../../../../docs/debt/).

> **Format, naming (`<topic>-<desc>.md`), the two-layer model (living static record + append-only Encounters ledger), priority levels, and escalation rules are authoritative in the workspace track** — see [`docs/debt/README.md`](../../../../docs/debt/README.md). The ledger is append-only: never edit a historic row.

Start a new debt item by copying [TEMPLATE.md](TEMPLATE.md). Log an encounter by appending a row to that item's Encounters table and bumping `Hits`.
